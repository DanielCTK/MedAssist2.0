# Thư mục Ảnh Tham Khảo & Thiết Kế (UI References)

Bạn hãy lưu các ảnh chụp màn hình, thiết kế, hoặc bài báo khoa học vào thư mục này để dễ quản lý.

## Cách để AI (tôi) code giao diện từ ảnh:

1. **Lưu ảnh** vào thư mục này (để bạn dễ tìm lại sau này).
2. **Kéo thả ảnh** từ thư mục này trực tiếp vào khung chat với AI.
3. **Gõ lệnh:** "Hãy code giao diện giống hệt ảnh này cho trang Dashboard" (hoặc trang bạn muốn).

**Lưu ý:** Tôi không thể tự đọc file trong thư mục này nếu bạn không gửi nó lên chat.
